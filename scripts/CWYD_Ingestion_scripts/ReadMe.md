# Azure Search Service Data Import & Export

## 1. Import Data into Azure Search Service

To import data into your Azure Search service:

1. Open the `import_data.ipynb` notebook.
2. Update the **resource group name** in the notebook with your target resource group.
3. Click the **Run All** button — it will automatically upload all the data into your Azure Search service.

---

## 2. Export Data from Azure Search Service

To export data from your Azure Search service:

1. Open the `export_data.ipynb` notebook.
2. Update the **resource group name** in the notebook with your source resource group.
3. Click the **Run All** button — it will automatically export all the data from your Azure Search service.
